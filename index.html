授权接口描述#

认证服务器接口地址

地址 说明
https://login.fitgreat.cn/oauth/v2/token 获取接入Token接口地址

请求参数，客户端发出的HTTP请求，包含以下参数：

     POST /token HTTP/1.1
     Host: server.example.com
     Content-Type: application/x-www-form-urlencoded

     grant_type=client_credentials&client_id=xxx&client_secret=xxx

响应参数，包含以下内容：

     HTTP/1.1 200 OK
     Content-Type: application/json;charset=UTF-8
     Cache-Control: no-store
     Pragma: no-cache

     {
       "access_token":"2YotnFZFEjr1zCsicMWpAA",
       "token_type":"access_token",
       "expires_in":3600
     }

查询接口描述#

接口地址

地址 说明
https://iot-test.fitgreat.cn/api/patient/data 根据床位ID获取床位对应的病人数据

请求参数，客户端发出的HTTP请求，包含以下参数：

     POST /patient/data HTTP/1.1
     Host: server.example.com
     Authorization: Basic czZCaGRSa3F0MzpnWDFmQmF0M2JW
     Content-Type: application/x-www-form-urlencoded

     bed_id=xxx

响应参数，包含以下内容：

     HTTP/1.1 200 OK
     Content-Type: application/json;charset=UTF-8
     Cache-Control: no-store
     Pragma: no-cache

     {
        "bed_id":"1-2", // 床位号 
        "ecgData":{ 
            "heartRate":106, // 心率 
            "breath":30, // 呼吸率 
            "timestamp":1581648799344 // 更新时间戳
        },
        "ecgState":{ 
            "battery":80, // 电量
            "falloff":false, // 脱落状态 
            "timestamp":1581646631942 // 更新时间戳
        },
        "tempData":{ 
            "temp":3874, // 体温值, 实际体温值需要除100, 例: 3874 -> 38.74
            "tempUpdateTime":1581756665000 // 更新时间戳 
        },
        "tempState":{ 
            "batteryLevel":90, // 电量 
            "chargeState":false, // 充电状态 
            "falloff":false, // 是否脱落      
            "fracture":false, // 是否断裂 
            "stateUpdateTime":1581756362670, // 状态更新时间 
        } 
     }
